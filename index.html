Koneksi Backend

Tempel URL deployment Web App dari Google Apps Script Anda (diakhiri /exec). Lihat panduan deploy di komentar paling atas file Code.gs.

Jika status "Tidak Terhubung", periksa: (1) deployment Apps Script sudah "Anyone" akses, (2) URL diakhiri /exec bukan /dev, (3) fungsi setupSheets sudah dijalankan sekali di editor Apps Script.

Cache Offline (SQLite)

Setiap data yang berhasil dimuat saat online otomatis disimpan ke database SQLite lokal (sql.js) di perangkat/browser ini, sehingga dashboard tetap bisa dibuka saat tidak ada internet. Aplikasi (tampilan) juga tersimpan otomatis lewat Service Worker agar bisa dibuka tanpa koneksi sama sekali setelah pertama kali online.
